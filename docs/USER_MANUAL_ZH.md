# SportsCounter 用户使用手册

## 1. 软件简介
SportsCounter 是一款家庭运动计数工具，支持：
- 手动计数（按钮或快捷键）
- 自动计数（定时累加）
- 多门限颜色提醒
- 多用户登录与独立配置管理

适合家人共用，同一台电脑每个用户可以保存自己的参数。

## 2. 第一次使用
1. 启动程序后进入登录窗口。
2. 默认管理员账号：
   - 用户名：`admin`
   - 密码：`admin`
3. 建议首次登录后进入“我的信息”修改密码。
4. 若需要给家人使用，管理员在“用户管理”中新增用户。

## 3. 主界面说明
主界面由 4 部分组成：
1. 顶部栏：当前用户、登录/登出、我的信息、配置管理（管理员可见用户管理）
2. 计数区：当前次数显示、当前配置名称
3. 快捷操作区：`+1`、`清零`
4. 配置区：热键、自动计数、门限颜色配置

## 4. 主要功能列表
1. 用户登录/登出
2. 计数与清零
3. 自动计数启停与间隔设置
4. 热键录制（支持清空自动计数热键）
5. 门限颜色规则管理
6. 多配置管理（新建/加载/设默认/删除）
7. 管理员用户管理（新增、编辑、删除）

## 5. 功能使用指导

### 5.1 手动计数
- 点击 `+1`：次数加一
- 点击 `清零`：次数重置为 0

### 5.2 自动计数
1. 在“自动计数配置”中设置间隔秒数（步长 0.5 秒）。
2. 点击 `启动计数`。
3. 运行中按钮会变为 `停止计数`。

### 5.3 热键配置
1. 点击“配置热键(弹窗)”。
2. 对应项点击“录制”，按下目标按键。
3. 自动计数热键可“清空”（避免与其他软件冲突）。

### 5.4 门限颜色配置
1. 点击“新增一项配置”。
2. 录入名称、门限值、颜色、说明。
3. 可选择列表项后“编辑选中项”。
4. 次数达到门限后，计数字体自动切换到对应颜色。

### 5.5 配置管理
1. 点击“配置管理”。
2. 可对当前用户配置执行：
   - 新的配置
   - 加载配置
   - 设为默认
   - 编辑信息
   - 删除配置
   - 保存当前配置

### 5.6 用户管理（仅管理员）
1. 管理员登录后显示“用户管理”按钮。
2. 在列表中选中用户，可执行“编辑用户/删除用户”。
3. 新增用户时填写用户名、初始密码、说明。

## 6. 数据存储位置
Windows 下数据自动保存到：
- `%APPDATA%\SportsCounter`

包含：
- `counter_app.ini`：系统与登录偏好
- `counter_app.db`：用户、配置、登录历史

## 7. 使用注意事项与建议
1. 首次使用后请修改 `admin` 密码。
2. 自动计数间隔不要设置过小，避免误触发或频繁增长。
3. 删除用户前确认其配置是否还需要保留。
4. 升级安装不会影响 `%APPDATA%\SportsCounter` 中的数据，但建议定期备份。

## 8. 常见问题
1. 每次都像首次使用：检查 `%APPDATA%\SportsCounter` 是否可写。
2. 登录失败：确认密码是否已修改；若修改过请重新输入。
3. 非管理员看不到用户管理：这是正常权限设计。
